# K-Means

**K-Means** — один из самых используемых алгоритмов кластеризации в production.

## Где K-Means используется в production

| Сценарий | Применение | Пример
|--|--|--
| Сегментация пользователей | Группировка по поведению | "Активные", "Спящие", "Новые" клиенты
| Рекомендательные системы | Кластеризация товаров/контента | Похожие фильмы, товары
| Сжатие данных (Vector Quantization) | Уменьшение размерности | Сжатие изображений, цветовые палитры
| Feature engineering | Создание новых признаков | "Расстояние до центра кластера"
| Аномалии | Точки далеко от центров = выбросы | Фрод-детекция
| Инициализация | Стартовые точки для других алгоритмов | K-Means++ для GMM

## Пример в production-коде

```py
import numpy as np
from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler

# Данные: активность пользователей (визиты, покупки, время на сайте)
X = np.array([
    [10, 2, 300],   # много визитов, мало покупок, много времени — "смотрящие"
    [2, 0, 10],     # редко заходят — "спящие"
    [50, 20, 600],  # активные покупатели
    [3, 1, 15],
    [45, 18, 550],
    [12, 3, 280],
])

# Масштабирование критично! (иначе "время" доминирует)
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# Плохо: n_init=1, random_state не задан → разные результаты каждый запуск
# K-Means: разделим на 3 сегмента
kmeans = KMeans(
    n_clusters=3,
    init='k-means++',      # умная инициализация (не рандом)
    n_init=10,             # запускаем 10 раз, выбираем лучший
    max_iter=300,
    random_state=42        # воспроизводимость
)
kmeans.fit(X_scaled)

# Результаты
labels = kmeans.labels_          # кластер для каждого пользователя
centers = kmeans.cluster_centers_  # центры кластеров

print(f"Сегменты: {labels}")       # [2 1 0 1 0 2]
print(f"Инерция (сумма расстояний): {kmeans.inertia_:.2f}")

# Для нового пользователя
new_user = np.array([[8, 1, 250]])
new_user_scaled = scaler.transform(new_user)
segment = kmeans.predict(new_user_scaled)
print(f"Новый пользователь в сегменте: {segment[0]}")
```

```
Сегменты: [2 1 0 1 0 2]
Инерция (сумма расстояний): 0.11
Новый пользователь в сегменте: 2
```

## Когда K-Means НЕ подходит

| Проблема | Почему | Альтернатива
|--|--|--
| Несферические кластеры | Ищет только "круглые" | DBSCAN, GMM
| Разная плотность | Один K для всех плотностей | HDBSCAN
| Категориальные признаки | Евклидово расстояние бессмысленно | K-Modes
| Большие данные (>100k) | Медленно сходится | MiniBatchKMeans
| Неизвестное K | Нужно заранее задавать | Иерархическая кластеризация

## MiniBatchKMeans для больших данных

```py
from sklearn.cluster import MiniBatchKMeans

# В 3-10x быстрее, почти такая же точность
mb_kmeans = MiniBatchKMeans(
    n_clusters=100,
    batch_size=1000,  # обрабатывает батчами
    random_state=42
)

# Потоковая обработка
for batch in data_generator():
    mb_kmeans.partial_fit(batch)  # online обучение
```
